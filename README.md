# Q1 What is the difference between null and undefined?
# Answer: undefined মানে ভ্যারিয়েবল আছে কিন্তু কোনো value দেওয়া হয়নি & null মানে ইচ্ছা করে value খালি রাখা হয়েছে 

# Q2 What is the use of the map() function in JavaScript? How is it different from forEach()?
# Answer: map() নতুন array তৈরি করে & forEach() শুধু loop চালায়, নতুন array দেয় না

# Q3 What is the difference between == and ===?
# Answer: == Type convert করে compare করে & === Type + value দুইটাই check করে 

# Q4 What is the significance of async/await in fetching API data? 
# Answer: async/await asynchronous code কে synchronous এর মতো readable করে 

# Q5 Explain the concept of Scope in JavaScript (Global, Function, Block)
# Answer: Global Scope সব জায়গা থেকে access করা যায়, Function Scope শুধু function এর ভিতরে & Block Scope { } এর ভিতরে
